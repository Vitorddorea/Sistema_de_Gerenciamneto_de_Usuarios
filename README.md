# 🧑‍💻 Sistema de Cadastro de Usuários (CLI)

Projeto desenvolvido em Python com foco em lógica de programação e simulação de operações de banco de dados (CRUD).

## 📌 Objetivo

Criar um sistema simples de gerenciamento de usuários via terminal, aplicando conceitos de:

- Estruturas de dados (listas e dicionários)
- Funções
- Validação de entrada
- Controle de fluxo
- Simulação de chave primária
- CRUD (Create, Read, Update, Delete)

---

## 🚀 Funcionalidades

✔ Cadastrar usuários  
✔ Listar usuários  
✔ Atualizar usuários  
✔ Deletar usuários  
✔ Geração automática de ID único  
✔ Validação de ano de nascimento  
✔ Tratamento de erro para entrada inválida  

---

## 🛠 Tecnologias Utilizadas

- Python 3
- Módulo `datetime`
- Módulo `os`

---

## 🧠 Conceitos Aplicados

- Simulação de banco de dados em memória
- Identificação única (ID semelhante a chave primária)
- Estrutura de menu interativo
- Organização modular com funções
- Tratamento de exceções com try/except

---

## ▶ Como Executar

1. Clone o repositório: https://github.com/Vitorddorea/Sistema_de_Gerenciamneto_de_Usuarios

3. Acesse a pasta do projeto: Sistema_de_Gerenciamneto_de_Usuarios

4. Execute: python main.py
